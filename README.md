subber
======

Subber is a subtitle resync tool

# usage

```
  usage: node index.js --f 3000 --input filename.srt {options}
  Options:

    -h, --help                output usage information
    -V, --version             output the version number
    -f, --forward [delay]     forward delay value
    -r, --rewind [delay]      rewind delay value
    -i, --input [filename]    the srt file to process
    -o, --output [filename]   the output filename
```
